# 游戏美术效果对比工具 - 技术方案

## 核心功能

1. **上传两张图**：美术效果图 + 游戏实机截图
2. **AI 分析对比**：从构图、配色、细节、氛围等维度评估差距
3. **输出报告**：差异点 + 改进建议

---

## 技术方案

### 方案 A：纯 Claude API（推荐起步）

```
用户上传图片 → Claude API (Vision) → 返回对比分析 → 展示结果
```

**优点**：简单直接，Claude 的视觉理解能力强  
**缺点**：每张图都调 API，成本累积

### 方案 B：本地预处理 + Claude

```
图片 → 本地算法(结构相似度/颜色直方图) → 量化指标 + 图片 → Claude → 综合报告
```

**优点**：混合分析，更专业  
**缺点**：开发量大一点
